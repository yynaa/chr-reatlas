# chr-reatlas & reatlaser

*a renderer/reatlaser for CHR tables in .nes roms*

![Crates.io Version](https://img.shields.io/crates/v/chr-reatlas?link=https%3A%2F%2Fcrates.io%2Fcrates%2Fchr-reatlas)
![docs.rs](https://img.shields.io/docsrs/chr-reatlas?link=https%3A%2F%2Fdocs.rs%2Fchr-reatlas%2Flatest%2Fchr_reatlas%2F)

![](assets/rockman.png)

## installing reatlaser

```sh
cargo install --path crates/reatlaser
```

### legacy cli

```sh
cargo install --path crates/reatlaser-cli
```

## crates overview

- [`chr-reatlas`](https://github.com/yynaa/chr-reatlas/tree/main/crates/chr-reatlas): rust library
- [`reatlaser-cli`](https://github.com/yynaa/chr-reatlas/tree/main/crates/reatlaser-cli): cli for reatlaser (legacy app)
- [`reatlaser`](https://github.com/yynaa/chr-reatlas/tree/main/crates/reatlaser): visual app for editing atlases
